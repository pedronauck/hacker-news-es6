## Hacker News ES6
Just a simple Hacker News Feed built using these ES6 features:

- Modules
- Classes
- Template Literals
- Let (Block scoping)
- Arrow Functions

##### Demo [here](http://pedronauck.github.io/hacker-news-es6)!

## Usabe

If you can see the application working, you can run:

```bash
  npm install
  gulp
```
